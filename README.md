# VT — Vibe-Trading 策略仓库

基于 Vibe-Trading 框架开发的量化选股策略，部署于国金 QMT 平台。

## 快速导航

📖 **[使用指南 → docs/使用指南.md](docs/使用指南.md)** — 项目完整说明文档

## 目录结构

| 目录 | 说明 |
|------|------|
| `docs/` | 项目文档和使用指南 |
| `策略脚本/` | QMT 平台策略（#coding:gbk） |
| `回测脚本/` | 回测与模拟验证代码 |
| `数据获取/` | 数据下载和处理工具 |
| `数据缓存/` | akshare 缓存的历史日线数据 |
| `股票池/` | 候选股票清单 |
| `策略提示词/` | Vibe-Trading 输入提示词 |
| `回测结果/` | 净值曲线、交易记录、指标报告 |
| `QMT平台规则相关/` | QMT API 封装资料 |

## 技术栈

- **策略开发**: Vibe-Trading (AI) + Python 3.11 (WSL)
- **数据源**: akshare / 东方财富数据中心
- **生产部署**: 国金 QMT 极速策略交易系统 (Python 3.6.8)
